
  # OnAirDeck

  This is a code bundle for OnAirDeck. The original project is available at https://www.figma.com/design/6YX8iZsE6p72c00Paq0YNQ/OnAirDeck.

  ## Running the code

  Use Node.js `>=20` (recommended: `nvm use`, based on `.nvmrc`).

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
